# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, safe, and respectful environment for everyone contributing to this project.

## Expected Behavior

- Be respectful and constructive in communication.
- Assume positive intent and give actionable feedback.
- Focus on what is best for the community and project.
- Accept responsibility, apologize when needed, and learn from mistakes.

## Unacceptable Behavior

- Harassment, personal attacks, intimidation, or discrimination.
- Trolling, insults, or inflammatory language.
- Publishing private information without explicit permission.
- Repeated disruptive behavior in issues, PRs, or discussions.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, discussions, and chat channels linked to the project.

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers via a private channel used by your team.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.
